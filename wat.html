Wat is Javascript?

Wat is Javascript

Javascript is een van de meest populaire en meestgebruikte programmeertalen in de wereld. Wie Javascript goed beheerst kan werken als een front-end, back-end en full-stack developer. Voor een lange tijd werd JS enkel gebruikt in browsers voor het bouwen van interactieve webpagina's. Vroeger noemde developers het een toy-language, maar die dagen zijn lang voorbij. Dankzij grote support en investering van grote bedrijven zoals Facebook en Google.

Hoe ontstond Javascript?

JavaScript is ontstaan bij Netscape Communications Corporation, een internetbedrijf dat in de jaren 90 de Netscape Navigator-webbrowser ontwikkelde. Brendan Eich, een softwareontwikkelaar bij Netscape, creÃ«erde JavaScript in slechts tien dagen in 1995 als een lichte scripttaal om webpagina's interactiever te maken. Het werd oorspronkelijk "LiveScript" genoemd, maar werd snel hernoemd naar "JavaScript" om de populariteit van Sun Microsystems' Java-taal te benutten. Later werd JavaScript gestandaardiseerd als ECMAScript en is het uitgegroeid tot een van de meest gebruikte programmeertalen voor webontwikkeling.

Waar draait Javascript?

Javascript-code is oorspronkelijk ontworpen om alleen in browsers te draaien. Elke browser heeft een javascript engine waarmee we javascript code kunnen uitvoeren. De javascript engine van Chrome heet V8, die van Firefox heet Spidermonkey. In 2009 nam Ryan Dhal de Chrome-engine en verwerkte die in een C++-programma dat hij Node noemde. Met Node kunnen we javascript-code uitvoeren buiten een browser, zodat we de backend voor onze web- en mobiele applicaties kunnen bouwen.
